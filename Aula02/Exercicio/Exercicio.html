1- Onde são descritas as regras CSS através do modo Internal?
Rta: No Modo Internal o CSS é escrito diretamente dentro do arquivo HTML, na tag style no head.
2 - Onde são descritas as regras CSS através do modo External?
Rta: Modo External é quando o CSS está em um arquivo separado e é vinculado ao HTML com a tag link.
3 - Qual a funcionalidade de uma classe CSS?
Rta: Uma classe CSS permite aplicar estilos. É definida com um ponto no CSS.
4 - Em quantos elementos HTML é possível aplicar CSS através do ID do elemento?
Rta: Um ID deve ser único por página HTML. Ou seja, apenas um elemento deve usar um determinado ID.
5 - De acordo com o trecho de códigos abaixo, qual será a cor da fonte? JUSTIFIQUE!
Rta: A cor será azul, pois o estilo inline tem maior prioridade que estilos externos ou internos.
6 - Qual a funcionalidade da regra padding?
Rta: A propriedade padding no CSS define o espaçamento interno de um elemento, ou seja, a distância entre o conteúdo do elemento e suas bordas.
7 - Qual o significado da regra padding: 0px 10px 0px 5px?
Rta: Essa é uma forma abreviada de definir o padding para os quatro lados do elemento.
8 - Qual a funcionalidade da regra margin?
Rta: A propriedade margin define o espaçamento externo de um elemento ou seja, a distância entre o elemento e os outros elementos ao redor.
9 - O trecho de código abaixo deverá dispor os elementos em linhas ou em colunas?
Rta: Os elementos serão dispostos em linha, ou seja, lado a lado na mesma linha, não em colunas.
